## 📋 نظرة عامة

**MedinteL** هي منصة ذكاء طبي حديثة مصممة لتحسين تجربة الرعاية الصحية من خلال توفير أدوات تشخيص ذكية، ومتابعة صحية شاملة، واستشارات طبية عن بُعد.

### 🎯 الهدف من المشروع

توفير حل تقني متكامل يربط بين المرضى ومقدمي الرعاية الصحية، مع التركيز على:
- **التشخيص الذكي** باستخدام تقنيات AI
- **المتابعة الصحية** المستمرة
- **الاستشارات عن بُعد** بسهولة وأمان
- **إدارة السجلات الطبية** بشكل آمن

---

## ✨ المميزات

### 🔍 التشخيص الذكي
- تحليل الأعراض باستخدام خوارزميات متقدمة
- اقتراحات تشخيصية أولية
- ربط مع قواعد بيانات طبية موثوقة

### 📊 المتابعة الصحية
- تتبع المؤشرات الحيوية
- تذكيرات بالأدوية والمواعيد
- تقارير صحية دورية

### 💬 الاستشارات عن بُعد
- حجز مواعيد مع أطباء متخصصين
- مكالمات فيديو آمنة
- وصفات طبية إلكترونية

### 📱 واجهة مستخدم حديثة
- تصميم responsive يعمل على جميع الأجهزة
- واجهة سهلة الاستخدام باللغة العربية
- تجربة مستخدم سلسة

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة بناء واجهات المستخدم
- **Vite** - أداة بناء وتطوير سريعة
- **Tailwind CSS** - إطار عمل CSS للتصميم
- **React Router** - التنقل بين الصفحات
- **Lucide React** - أيقونات حديثة

### الأدوات
- **ESLint** - فحص جودة الكود
- **Git** - إدارة الإصدارات
